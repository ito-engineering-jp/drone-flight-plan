# ドローン飛行計画書 作成ツール

フォームに入力するだけで、クライアント提出用のドローン飛行計画書PDFを即時作成できるWebツールです。

---

## 機能

- **8セクション構成のフォーム**（基本情報・撮影概要・飛行日時場所・航空法・機体操縦者・許可申請・保険・緊急連絡）
- **国土交通省 許可書PDFの読み込み**（許可番号・有効期間・飛行経路・機体一覧を自動入力）
- **機体一覧の選択読み込み**（別紙PDFから読み込む機体を個別に選択可能）
- **案件管理**（複数案件の保存・切り替え・複製・削除）
- **定型情報プリセット**（会社情報・操縦者・機体・許可情報などを登録して呼び出し）
- **国土地理院地図**（撮影地点ごとにLeaflet地図を表示）
- **PDF印刷出力**（ブラウザの印刷機能でA4 PDF出力）
- **スマートフォン対応**

---

## 使い方

### PCで使う（ローカル）

```bash
# このリポジトリをクローン or HTMLファイルをダウンロード
git clone https://github.com/ito-engineering-jp/drone-flight-plan.git
cd drone-flight-plan

# ローカルサーバーを起動（インターネット接続が必要）
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080/drone_flight_plan.html` を開く。

### スマートフォン・どこからでも使う（GitHub Pages）

以下のURLにアクセスするだけで使えます：

**https://ito-engineering-jp.github.io/drone-flight-plan/drone_flight_plan.html**

---

## 注意事項

- **インターネット接続が必要**です（地図表示・PDF読み込み機能でCDNを使用）
- 案件データはブラウザの **localStorage** に保存されます。別のブラウザ・端末には引き継がれません
- PDF印刷出力はブラウザの印刷機能を使用します。ChromeまたはEdge推奨
